## Impaciente?

A maneira mais rápida de criar e executar um projeto Start é com a CLI. Basta executar

```
pnpm create @tanstack/start@latest
```

ou

```
npm create @tanstack/start@latest
```

dependendo do gerenciador de pacotes de sua preferência. Você será solicitado a adicionar coisas como Tailwind, eslint e diversas outras opções.

Você também pode clonar e executar o exemplo [Basic](https://github.com/TanStack/router/tree/main/examples/react/start-basic) imediatamente com os seguintes comandos:

```bash
npx gitpick TanStack/router/tree/main/examples/react/start-basic start-basic
cd start-basic
npm install
npm run dev
```

Se preferir usar um exemplo diferente, substitua `start-basic` acima pelo slug do exemplo que deseja usar da lista abaixo.

Depois de clonar o exemplo desejado, volte ao guia de [Routing](./guide/routing) para aprender a usar o TanStack Start!

### Stackblitz

Cada exemplo acima possui uma pré-visualização embutida no Stackblitz para encontrar aquele que pareça um bom ponto de partida.

### Deploy Rápido

Para fazer deploy rápido de um exemplo, clique no botão **Deploy to Netlify** na página do exemplo para clonar e fazer deploy do exemplo no Netlify.

### Deploy Manual

Para clonar e fazer deploy do exemplo manualmente em qualquer outro lugar, use os seguintes comandos substituindo `EXAMPLE_SLUG` pelo slug do exemplo que deseja usar da lista acima:

```bash
npx gitpick TanStack/router/tree/main/examples/react/EXAMPLE_SLUG my-new-project
cd my-new-project
npm install
npm run dev
```

Depois de clonar ou fazer deploy de um exemplo, volte ao guia de [Routing](./guide/routing) para aprender a usar o TanStack Start!
