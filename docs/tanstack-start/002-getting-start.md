## Inicie um novo projeto do zero

Escolha uma das seguintes opcoes para comecar a construir um _novo_ projeto TanStack Start:

- [TanStack Start CLI](./quick-start) - Basta executar `npm create @tanstack/start@latest`. Local, rapido e opcionalmente customizavel
- [TanStack Builder](https://tanstack.com/builder) - Uma interface visual para configurar novos projetos TanStack com poucos cliques
- [Exemplos de Quick Start](./quick-start) Baixe ou clone um dos nossos exemplos oficiais
- [Construa um projeto do zero](./build-from-scratch) - Um guia para construir um projeto TanStack Start linha por linha, arquivo por arquivo.

## Proximos Passos
