## Iniciar um novo projeto do zero

Escolha uma das seguintes opções para começar a construir um _novo_ projeto TanStack Start:

- [TanStack Start CLI](./quick-start) - Basta executar `npm create @tanstack/start@latest`. Local, rápido e opcionalmente personalizável
- [TanStack Builder](#) (em breve!) - Uma interface visual para configurar novos projetos TanStack com poucos cliques
- [Exemplos de Início Rápido](./quick-start) Baixe ou clone um dos nossos exemplos oficiais
- [Construir um projeto do zero](./build-from-scratch) - Um guia para construir um projeto TanStack Start linha por linha, arquivo por arquivo.
