# maiyu:all — All Skills

When this skill is activated, identify whether the task is **backend**, **frontend**, or **devops**, then invoke the corresponding skill:

- **Backend tasks** — Use `maiyu:backend`
- **Frontend tasks** — Use `maiyu:frontend`
- **DevOps tasks** — Use `maiyu:devops`

For compound tasks that span multiple areas (e.g., "create a full CRUD"), invoke the relevant skills in sequence.
